# Story time

Read a story out loud

```
Play a AI generated story as audio

Usage: story-time [OPTIONS] --chat-gpt-key <CHAT_GPT_KEY> --elevenlabs-key <ELEVENLABS_KEY> --prompt <PROMPT>

Options:
  -c, --chat-gpt-key <CHAT_GPT_KEY>
          Key for chatgpt [env: CHAT_GPT_KEY=]
  -e, --elevenlabs-key <ELEVENLABS_KEY>
          Key for elevenlabs [env: ELEVENLABS_KEY=]
  -p, --prompt <PROMPT>
          Prompt [env: PROMPT=]
  -s, --style <STYLE>
          A style to read in [env: STYLE=] [default: "You are reading aloud"]
  -v, --voice <VOICE>
          [env: VOICE=] [default: MF3mGyEYCl7XYWbV9V6O]
  -o, --output <OUTPUT>
          Save to a file rather than reading aloud [env: OUTPUT=]
  -h, --help
          Print help
  -V, --version
          Print version
```
